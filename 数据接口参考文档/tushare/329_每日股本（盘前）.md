# 每日股本（盘前）

来源: https://tushare.pro/document/2?doc_id=329

---

## 接口基本信息
- **接口名称**：`stk_premarket`
- **功能描述**：每日开盘前获取当日股票的股本情况，包括总股本和流通股本，涨跌停价格等
- **数据限制**：单次最多8000条记录，支持循环获取
- **权限要求**：需单独申请开权限，与积分无关

## 输入参数
该接口接收可选参数：
- `ts_code`（字符型）：股票代码
- `trade_date`（字符型）：交易日期，YYYYMMDD格式
- `start_date`（字符型）：开始日期
- `end_date`（字符型）：结束日期

## 输出参数
返回以下七个字段：

| 字段名 | 数据类型 | 含义 |
|--------|--------|------|
| trade_date | 字符型 | 交易日期 |
| ts_code | 字符型 | TS股票代码 |
| total_share | 浮点型 | 总股本（万股） |
| float_share | 浮点型 | 流通股本（万股） |
| pre_close | 浮点型 | 昨日收盘价 |
| up_limit | 浮点型 | 今日涨停价 |
| down_limit | 浮点型 | 今日跌停价 |

## 调用示例
```python
pro = ts.pro_api()
df = pro.stk_premarket(trade_date='20240603')
```
