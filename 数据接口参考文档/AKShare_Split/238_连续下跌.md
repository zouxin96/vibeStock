#### 连续下跌

接口：stock_rank_lxxd_ths

目标地址：https://data.10jqka.com.cn/rank/lxxd/

描述：同花顺-数据中心-技术选股-连续下跌

限量：单次返回所有数据

输入参数

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

输出参数

| 名称    | 类型      | 描述      |
|-------|---------|---------|
| 序号    | int64   | -       |
| 股票代码  | object  | -       |
| 股票简称  | object  | -       |
| 收盘价   | float64 | 注意单位: 元 |
| 最高价   | float64 | 注意单位: 元 |
| 最低价   | float64 | 注意单位: 元 |
| 连涨天数  | int64   | -       |
| 连续涨跌幅 | float64 | 注意单位: % |
| 累计换手率 | float64 | 注意单位: % |
| 所属行业  | object  | -       |

接口示例：

```python
import akshare as ak

stock_rank_lxxd_ths_df = ak.stock_rank_lxxd_ths()
print(stock_rank_lxxd_ths_df)
```
