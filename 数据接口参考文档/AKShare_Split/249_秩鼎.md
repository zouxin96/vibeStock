接口: stock_esg_zd_sina

目标地址: https://finance.sina.com.cn/esg/grade.shtml

描述: 新浪财经-ESG评级中心-ESG评级-秩鼎

限量: 单次返回所有数据

| 名称  | 类型  | 描述  |
|-----|-----|-----|
| -   | -   | -   |

输出参数

| 名称     | 类型     | 描述 |
|--------|--------|----|
| 股票代码   | object | -  |
| ESG评分  | object | -  |
| 环境总评   | object | -  |
| 社会责任总评 | object | -  |
| 治理总评   | object | -  |
| 评分日期   | object | -  |


接口示例

```python
import akshare as ak

print(stock_esg_zd_sina_df)
```
