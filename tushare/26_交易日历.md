# 交易日历

来源: https://tushare.pro/document/2?doc_id=26

---

## 接口基本信息
- **接口名称**：`trade_cal`
- **接口描述**：获取各大交易所交易日历数据，默认提取的是上交所
- **所需积分**：2000积分
- **调试工具**：可通过数据工具调试和查看数据

## 输入参数

| 参数名 | 数据类型 | 必选 | 说明 |
|--------|---------|------|------|
| exchange | str | N | 交易所代码(SSE上交所、SZSE深交所、CFFEX中金所等) |
| start_date | str | N | 开始日期(格式：YYYYMMDD) |
| end_date | str | N | 结束日期(格式：YYYYMMDD) |
| is_open | str | N | 筛选条件：'0'休市 或 '1'交易 |

## 输出参数

| 参数名 | 数据类型 | 默认显示 | 说明 |
|--------|---------|---------|------|
| exchange | str | Y | 交易所代码 |
| cal_date | str | Y | 日历日期 |
| is_open | str | Y | 是否交易 0休市 1交易 |
| pretrade_date | str | Y | 前一个交易日 |

## 调用示例

```python
pro = ts.pro_api()
df = pro.trade_cal(exchange='', start_date='20180101', end_date='20181231')

# 或使用query方法
df = pro.query('trade_cal', start_date='20180101', end_date='20181231')
```

## 数据示例
返回DataFrame格式，包含exchange、cal_date、is_open三列信息，展示各交易日的开闭市状态。
