# vibeStock 项目文件结构说明

本项目主要用于收集和整理 Tushare 财经数据接口的相关文档和脚本。

## 目录结构

### 根目录

*   `README.md`: 本文件，项目结构说明。
*   `tushare/`: 核心数据文件夹，包含各类财经数据的接口文档（Markdown 格式）以及相关的爬虫脚本和索引文件。

### tushare 目录
